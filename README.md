# Landing Page

This project is part of **The Odin Project** curriculum.  
The goal was to recreate a landing page design using HTML and CSS, focusing on layout and styling.

## Project Description

This landing page includes a hero section, information cards, a testimonial section, and a call-to-action area.  
The layout is built to closely match the provided design while maintaining clean and readable code.

## What I Learned

Through this project, I practiced:
- structuring a full webpage with semantic HTML
- styling layouts using CSS
- using Flexbox for alignment and spacing
- working with fonts, colors, and spacing
- breaking a design into reusable sections
- improving visual hierarchy and readability

## Technologies Used

- HTML5
- CSS3 (Flexbox)
